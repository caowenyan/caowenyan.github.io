搭建hexo博客

  • hexo

准备工作

  1. github账号
  2. node.js、npm环境
  3. git环境

    搭建github环境

  4. 创建github仓库，名称必须是你的用户名.github.io，将来访问的地址是：http://你的用户名.github.io
  5. 配置SSH key
  • 将本地的ssh公钥加到github的SSH and GPG keys上
  • 测试配置是否成功
    ssh -T git@github.com
    提示：Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.则说明配置成功。
